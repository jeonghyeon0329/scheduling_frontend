# scheduling backend
스케줄링 시스템 프론트 개발
![메인 페이지](./public/images/title-page.png)

## 📦 요구 사항
- node v22.16.0
- npm v10.9.2

## 🛠 설치
```bash
# 1) 저장소 클론
git clone https://github.com/jeonghyeon0329/scheduling_frontend.git
cd scheduling_frontend

# 2) 의존성 설치
npm install

# 3) 개발 서버 실행
npm start
```
